# Animation Configuration File
# 動畫設定檔
# 
# 使用說明：
# 1. 修改這個檔案可以即時調整動畫效果
# 2. 參數範圍：
#    - 平滑度參數(smooth): 0.0-1.0，值越大變化越快
#    - 持續時間(duration): 正整數，表示幅畫面數
#    - 透明度(alpha): 0-255，0完全透明，255完全不透明
#    - 比例參數(ratio): 0.0-1.0，表示相對大小
#    - 線條粗細(thickness): 正整數，像素數
# 3. 修改後儲存檔案，程式會在下次檢測到人臉時套用新設定
#
# 四個動畫狀態說明 (已優化為更穩定版本)：
# 狀態1: 外框角落線條逐漸出現 (40幅畫面) - 框尺寸放大1.5倍
# 狀態2: 內框半透明區域出現 (40幅畫面) - 保持之前效果
# 狀態3: 十字準星線條從中心開始延伸 (20幅畫面) - 保持之前效果  
# 狀態4: 十字準星線條完全延伸到邊緣 (30幅畫面) - 不會重置，保持穩定
#
# 💡 新特性：
# - 檢測框尺寸增大1.5倍，更明顯
# - 動畫不會輕易重置，更穩定
# - 使用累積狀態，後續狀態保持前面狀態的效果
# - 優化平滑度參數，減少抖動
